HANGMAN

A simple, simple hangman game.

View Github

View Project
  • HTML
  • CSS
  • Vanilla Javascript

This was the first project I created outside of following tutorials on Youtube and Udemy. The first project I had gone off on my own to complete without anyone holding my hand, I was unsure whether or not I would be able to pull it off, but i thoroughly enjoyed it. I remember being at work and thinking through the problems I was encountering while making the game, excited to get home to implement the solutions I had come up with. This was refreshing because up until then I had been doing tutorial after tutorial, which was draining, and it didn’t feel like I was learning anything. This was what I now know as tutorial hell.

I built the core game logic first, before the UI, outputting everything to the console. I decided to have two game modes, PVP and PVE. PVE would require learning how to use and process HTTP requests to import a list of words, which at the time was far beyond what I knew how to do. So I focused on the PVP mode first and added PVE mode once the project was complete. With the UI I decided to keep things simple with a black and white minimalist design. The UI turned out to be the biggest challenge despite its simplicity. Because I didn’t know anything about managing reusable components, modals, or UI systems. I made it all up as I went along.
